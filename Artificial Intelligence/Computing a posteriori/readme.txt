Programming language : JAVA

Structure of the code:
The program reads the first argument which is the sequence of onservations, which is then used to count the numbers of L and C values in that sequence.
The possible hypothesis, prior probability values are stored in arrays of type double.
The probability of the next candy being picked to be a cherry candy and the probability of the next candy being picked to be a lime candy is then calculated.
The observations are written into a file named "result.txt".
